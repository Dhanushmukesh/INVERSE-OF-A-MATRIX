### Step 4: End the program
## Program:
```
#Program to find the inverse of a matrix.
#Developed By : Dhanush.G
#Register No: 2305002006

import numpy as np
A=np.array([[2,1,1],[1,1,1],[1,-1,2]])
inverse=np.linalg.inv(A)
print(inverse)
```
